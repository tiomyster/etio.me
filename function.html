FDM DEMO BLOCK

Summer 2013

Function Engineering Internship

overview

As part of my summer internship at Function Engineering, I created demo blocks and plaques that showed actual dimensions of features made with the FDM printer and the laser cutter. These trinkets could potentially be used as a reference when designing parts to be made on these machines. From this project, I became more familiar with the FDM printer and laser cutterand gained a sense of their capabilities and limitations.
