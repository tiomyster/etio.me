ORGANIC CRAYONS

MIT January 2013

D-Lab Development

overview

As a wrap up for the D-Lab: Development class, I visited Avani, a voluntary organization in the Kumaon region of Uttarakhand. Avani uses local pigments to dye yarn to be woven into a line of textile. At Avani. The organic crayon project is an expansion of Avani's product line. I worked on expanding the color palette for the beeswax-based crayons as well as developing a recipe for clay-based pastels. This project was very hands on and required a lot of experimentation, but it was rewarding to successfully make crayons and pastels and draw with them. A large part of this project was documentation so that next year's visitors would be able to pick up where I left off.
